## 카테고리별 프롬프트 찾기

### 1. 사진 스타일 / 포토리얼리즘
- 1990s/2000s 카메라 스타일
- 실리콘밸리 스타일 비즈니스 사진
- 빈티지 필름 감성 (Kodak Portra)
- 패션 에디토리얼

### 2. 3D / 피규어 스타일
- Funko Pop 피규어
- 치비(Q버전) 캐릭터
- LEGO 스타일
- 액션 피규어
- 스노우 글로브

### 3. 일러스트레이션
- 지브리 스타일
- Pixar 3D 스타일
- 미니멀리스트 일러스트
- 인포그래픽 카드
- 만화/웹툰 스타일

### 4. 창의적 편집
- 나이 변화 시뮬레이션
- 헤어스타일 변경
- 의상 변경
- 배경 교체
- 사진 복원/컬러화

### 5. 제품/마케팅
- 제품 광고 이미지
- 매거진 커버
- 소셜 미디어용 이미지
- 명함 디자인

## 프롬프트 요청 방법

저에게 다음과 같이 요청하시면 됩니다:

### 예시 요청

**1. 카테고리로 요청:**
```
"3D 피규어 스타일 프롬프트 5개 알려줘"
"빈티지 사진 스타일 프롬프트 추천해줘"
```

**2. 특정 목적으로 요청:**
```
"인물 사진을 비즈니스 프로필 사진으로 바꾸는 프롬프트"
"제품 광고용 프롬프트 모음"
```

**3. 스타일 참고 요청:**
```
"지브리 스타일 프롬프트 보여줘"
"Funko Pop 스타일로 만드는 프롬프트"
```

**4. 일괄 요청:**
```
"이미지 생성 프롬프트 10개 일괄 제공해줘"
"다양한 스타일의 프롬프트 모음 보여줘"
```
